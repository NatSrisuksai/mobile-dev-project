## Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd project-v1
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the project:**
   ```bash
   npx expo start
   ```
